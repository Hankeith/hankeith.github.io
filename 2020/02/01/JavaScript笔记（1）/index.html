从零学习的JavaScript

这部分的学习内容与资料都在W3school上，有任何问题都能在文中贴出的链接上得到解答。由于在本次笔记中的JavaScript实践都为交互式，不方便截图，故不放实践图了。

从实例开始学习

JavaScript 改变HTML内容

示例1

查找元素并将内容更改

利用getElementById()来完成

document.getElementById("kagura").

<p id="mea">JavaScript 改变 HTML 内容演示</p>

<button  onclick="document.getElementById('mea').innerHTML 
    = '内容已经改变力'"> 点击改变 </button>

注：JavaScript中双引号和单引号都被接受，但书写代码时一定要形成习惯，不然会犯一些低级却难发现的错误（）中的id名要用单引号'括起

JavaScript 改变HTML属性

示例2

查找元素并将图片更改

同样利用getElementById()来完成

<img id="aqua" border="0" src="**/**" style="text-align:center">
<button onclick="document.getElementById('aqua').src='***/***'">setu1</button> 
<button onclick="document.getElementById('aqua').src='*/*'">setu2</button>

JavaScript 改变HTML样式（CSS）

示例3
<p id="minato">JavaScript 改变 HTML 元素样式</p>
<button type="button" onclick="document.getElementById('demo').style.fontSize='35px'">
点击后文字的样式出现变化
</button>

那么，控制HTML元素的显示和隐藏应该也是控制元素样式的一部分

<p id="nana">JavaScript 能够改变 HTML 元素的样式。</p>

<button type="button" onclick="document.getElementById('nana').style.display='none'">
点击我！
</button>
<p id="fubuki" style="display:none">wait display</p>

<button type="button" onclick="document.getElementById('fubuki').style.display='block'">
点击显示
</button>
