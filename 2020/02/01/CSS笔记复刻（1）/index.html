层叠次序

当同一个 HTML 元素被不止一个样式定义时，会使用哪个样式呢？

一般而言，所有的样式会根据下面的规则层叠于一个新的虚拟样式表中，优先级随数字增大而增大

1.浏览器缺省设置

2.外部样式表

3.内部样式表（位于 <head>标签内部）

4.内联样式（在 HTML 元素内部）

因此，内联样式（在 HTML 元素内部）拥有最高的优先权，这意味着它将优先于以下的样式声明：<head> 标签中的样式声明，外部样式表中的样式声明，或者浏览器中的样式声明（缺省值）。

CSS语法

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明。

选择器通常是您需要改变样式的 HTML 元素。

每条声明由一个属性和一个值组成。

属性（property）是您希望设置的样式属性（style attribute）。每个属性有一个值。属性和值被冒号分开。

示例（如图）

（注：若值为多个单词，要给值加上引号）

body {
  color: #000;
  background: #fff;
  margin: 0;
  padding: 0;
  font-family: Georgia, Palatino, serif;
  }

选择器的分组和继承

分组

h1,h2,h3,h4,h5,h6 {
  color: green;
  }

继承

一般来说，子元素将从父元素继承属性

派生选择器

根据元素的上下文位置关系来定义样式

比方说，令h1中的 em为红色

<style type="text/css"
h1 em{color:red;}
</style>

id选择器

为标有特定id的HTML元素指定特定样式

id选择器以#定义

例1
#color1 {color:blue;}
#color2 {color:purple;}

于是乎下列id属性为color1的p元素显示为蓝色，id属性为color2的p元素显示为紫色

<p id="color1">
    显示为蓝色
</p>
<p id="color2">
    显示为紫色
</p>

id属性的值不能重复
CSS中

错误示范

#red{color:red;} <P id="red">这段字体是红的</p> <p id="red">这段也是红的</p>

这样是不可以的

要写成类：
.red {color:red;} <P class="red">这段字体是红的</p> <p class="red">这段也是红的</p>

实际应用的时候，ID一般会用来给有单独特性的标签来使用，而不会像上例这样使用，这样的情况，一般都会使用class，ID和Class的区别就在于此，设计这两个的初衷也是因为这样吧。

类选择器

类选择器以一个单号为标示

.center {text-align: center}

在该例中，所有拥有center类的HTML元素显示为居中

<h1 class="center">
标题居中
</h1>

<p class="center">
内容居中
</p>

在上面的代码中，h1和p元素都都有 center 类，两者都遵守 center选择器中的规则。

注意，类名的第一个字符不能是数字

对比

.fancy td {
    color: #f60;
    background: #666;
    }
td.fancy {
    color: #f60;
    background: #666;
    }

前者是类名为fancy的元素内部的表格单元会拥有这个样式，后者是表格单元中类名为fancy的会拥有这个样式

创建CSS

插入样式表

插入样式表的方法有三种

外部样式表

当样式需要应用于很多页面时，外部样式表比较方便。在使用外部样式表的情况下，你可以通过改变一个文件来改变整个站点的外观。每个页面使用 <link> 标签链接到样式表,<link>标签在（文档的）头部

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css" />
<link rel="stylesheet" type="text/css" href="mystyle.css" />
</head>

浏览器会从文件 mystyle.css 中读到样式声明，并根据它来改变文档样式。

外部样式表中不含有html标签并以.css扩展名进行保存

h {color: sienna;}
p {margin-left: 20px;}
body {background-image: url("../..");}

注：不要在属性值与单位之间留空格，例如20_px，这样可能会出错

内部样式表

当单个文档需要特殊的样式时，就应该使用内部样式表。可以使用 <style>标签在文档头部定义内部样式表，就像这样:

<head>
<style type="text/css">
  h {color: sienna;}
  p {margin-left: 20px;}
  body {background-image: url("../..");}
</style>
</head>

内联样式

一般用于当样式仅需要在一个元素上应用一次时。要使用内联样式，你需要在相关的标签内使用样式（style）属性。style 属性可以包含任何 CSS 属性。本例展示改变段落的上外边距和左外边距：

<p style="margin-tio:50px; margin-left: 20px">
This is a paragraph
</p>
