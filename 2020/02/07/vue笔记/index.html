vue笔记
note vue
vue
发布日期:   2020-02-07

在页面上安装vue只需在head内引入<script src="vue.js">

vue.js的应用可以分为两份部分，视图和脚本。

首先在页面上创建id为app的div标签，然后用左右双大括号的方式声明变量，

当引入一个vue.js时会声明一个全局变量Vue

一般来说

先建立一个id选择器

<div id="mea">
 {{message}}
 </div>

再构建vue脚本

由于vue是响应式，它的数据的获取和修改可以很方便的进行，又因为它是声明式范式(不同于原始js的命令式)，只要我们给它指明要求它就能很好地执行。

在学习vue的过程中要记住一些重要的指令

示例：Vue列表显示

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title></title>
<script src="vue.js" type="text/javascript" charset="utf-8"></script>
</head>
<body>
    <div id="app">
       <ul>
              <li v-for="item in vtb">{{item}}
              </li>
          </ul>
    </div>

    <script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
             vtb: ['mea','minato','kagura']
        }
    });
    </script>

</body>
</html>

这里用到了v-for循环指令

在这里若要在向列表中添加数据可以使用console中输入aqua.vtb.push(‘fubuki’)

示例：Vue点击计数

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <script src="vue.js">

        </script>
        <title></title>
    </head>
    <body>
        <div id="app">
            当前计数为{{counter}}

        <!--<button v-on:click="counter++">+</button>
        <button v-on:click="counter--">-</button>-->
    <button v-on:click="add">+</button>
    <button v-on:click="sub">-</button>
        </div>
    <script>
        let app = new Vue({
            el: '#app',
            data: {
                counter:0
            },
            methods: {
                add: function () {
                    console.log('add');
                    this.counter++/*注意在这一步counter前要用this，不然他会在全局去寻找这个个目标，或者用app.counter指定为app下的counter*/
                },
                sub: function () {
                    console.log('sub');
                    this.counter--
                }            
                      }    
        })
    </script>
    </body>
</html>

这里用到了v-on指令，即在监听某个事件，后面要给出对应的描述

语法糖：v-on:click可简写为@click

绑定属性

前面学习的指令主要讲值插入到模板的内容当中，但是除了内容外，许多属性也需要动态绑定，例如：

  • 动态绑定<a>元素的href属性
  • 动态绑定<img>元素的src属性

这个时候就要用v-bind指令

v-bind用于绑定一个或多个属性值，或者向另一个组件传递props值

例如通过data绑定元素的src和href

<div id="app">
<a v-bind:href="link">Vuejs</a>
<img v-bind:src="logoURL" alt="vuelogo">
</div>
<script src="vue.js">
  let app = new Vue({
    el:'#app'
    data: {
      logoURL: 'https://vuejs.org/images/logo.png',
      link: 'https://vuejs.org/'
    }
  })
</script>
